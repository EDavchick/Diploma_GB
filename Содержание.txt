Обзор популярных жанров онлайн-игр, анализ игр на Java и JavaFX
Изучение существующих многопользовательских 2D игр.
Анализ популярных платформ и инструментов для разработки игр на Java и JavaFX.
Исследование сетевых протоколов и технологий, необходимых для реализации многопользовательского взаимодействия.
Основные особенности и преимущества использования Java для разработки игр.
JavaFX: платформа для создания графических интерфейсов
Введение в JavaFX.
Основные компоненты JavaFX (сцены, узлы, макеты и т.д.).
Преимущества использования JavaFX для разработки игр.
Архитектура многопользовательских онлайн-игр
Основные компоненты многопользовательских игр (сервер, клиент, сетевые протоколы).
Принципы разработки серверной и клиентской частей.
Проблемы и решения, связанные с синхронизацией данных и производительностью.
Сравнительный анализ технологий 
Сравнение Java и других языков программирования для разработки игр.
Сравнительный анализ других платформ для создания интерфейсов (Swing, AWT и т.д.).
